01. What is LLM

Probability view: An LLM models the next‑token distribution \(P(x_{t+1} \mid x_1, x_2, \ldots, x_t)\). It assigns a probability to each possible next word given the previous context, and generation samples or selects the most likely token step by step.

1. Definition

A large language model (LLM) is a neural network trained to model the probability of sequences of tokens. It learns to generate, summarize, translate, and reason over text by predicting the next token given a context.

2. How It Is Trained

  • Pretraining: The model is trained on massive text corpora with self‑supervised learning (next‑token prediction).
  • Fine‑tuning: It may be adapted to specific tasks or domains with supervised data.
  • Alignment: Techniques like instruction tuning or RLHF can make outputs more helpful and safe.

3. What It Is Based On

  • Transformer architecture: Attention layers enable long‑range dependency modeling.
  • Tokenization: Text is converted into subword tokens.
  • Optimization: Training uses gradient descent with large‑scale distributed compute.

4. Applications

  • Chatbots and assistants
  • Search and question answering
  • Summarization and content generation
  • Code generation and debugging assistance
  • Translation and localization
