教育背景

  • 2022 年毕业于滑铁卢大学，获得 数学学士 学位，主修 数理金融，辅修 统计学

  • 我的学术经历主要集中在 人工智能、数量金融、统计学、计算机科学 和 纯数学，并在概率论、线性模型和算法思维方面打下了坚实基础

  • 目前我在 香港大学 攻读 人工智能理学硕士，课程包括 机器学习、深度学习、自然语言处理、数据挖掘与文本分析，以及 交易与金融中的机器学习

工作经历

全职数据分析师 — RBC Insurance 加拿大

  • 使用 Python 和 VBA 开发自动化可执行程序，优化审计、客户、次级保险公司报告及养老金管理流程， 平均每月减少 15 天的手动操作时间

  • 通过高级 Excel VBA 宏进行数据清洗，并结合 Python 脚本和 MS Access 进行数据管理与验证， 每个周期将年金数据实施时间缩短 2 天

  • 改进并执行复杂 SQL 脚本，使用 Python（Xlwings、Pandas、NumPy）验证数据完整性， 确保数据准确迁移及系统可靠性

  • 开发动态、交互式 Tableau 仪表盘，用于每月及临时业务和退休人员统计报告， 支持 VP 及精算团队的战略决策和有效年金定价

大宗商品研究实习生 — 郑州商品交易所期货与衍生品研究院 郑州

  • 使用 Python（PySpark、Requests、SQLAlchemy、NumPy）和 AWS Glue 分析并计算期货指数， 构建数据管道以处理大量交易数据。使用 Power BI 可视化研究结果，以确定年度研究主题。

  • 撰写动量交易研究报告，分析期货价格与策略渗透率之间的相关性， 以识别郑州商品交易所交易期货潜在的指数波动。

  • 执行数据清洗，包括删除重复或无关观测值、修复结构错误（拼写错误或大小写不正确）、筛除异常值、处理缺失数据，并进行验证。

  • 向利益相关者展示研究结果，将复杂洞见通过数据故事和 Tableau 可视化转化为可执行的建议。

金融分析师 - 中原证券

  • 与团队协作开展上交所二级市场研究， 分析不同行业板块中的潜在优质股票， 并协助撰写面向外部客户的投资研究报告

  • 评估客户的投资预算与风险承受能力， 提供个性化投资建议并推荐合适的金融产品

  • 在路演与投资汇报中， 以通俗易懂的方式向客户讲解金融分析逻辑， 与团队共同完成最优投资组合的推荐， 展现了良好的沟通能力与团队协作精神
